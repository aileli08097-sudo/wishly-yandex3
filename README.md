# Wishly
Wishly - это сайт, который помогает пользователям создавать списки желаемых подарков к праздникам и делиться ими с близкими. Веб-сайт решает проблему "что подарить" и позволяет избежать повторения подарков благодаря системе резервирования.

Пользователи могут создавать вишлисты под разные праздники, добавлять подарки с фото и ссылками на товары, настраивать уровень приватности, а также получать уведомления о резервировании подарков. Встроенный календарь праздников помогает не забыть о важных датах близких людей.

## Техническое задание
При создании сайта будет использоваться Flask, управление базой данных, работа с файлами, реализуется система регистрации, авторизации и уведомлений пользователям. После перехода на сайт пользователь должен будет зарегестрироваться или войти в аккаунт, после он сможет просматривать доступные вишлисты близких, создавать или редактировать свои. Можно будет отмечать дату праздника(чтобы приходили уведомления), добавлять ссылки и фотографии к желаемым подаркам.

## Пояснительная записка
Веб-сайт расположен по ссылке https://wishly-lf3b.onrender.com/
(Возможна долгая загрузка из-за продолжительной неактивности сайта, пожалуйста, подождите)

Для установки всех библиотек введите в терминал:
```shell
pip install -r requirements.txt
```
Сайт Wishly является веб-сервисом для создания и размещения списков желаний(вишлистов), которыми можно делиться с близкими без возможности узнать, кто какой подарок выберет. При переходе на главную страницу отображется либо список забронированных и созданных списков, либо сообщение об отсутствии таких списков. Есть регистрация и авторизация, уникальность никнеймов и электронных почт, бронирование конкретных желаний и отказ от бронирования, возможность обменивания вишлистами по ссылке, добавлени ссылок на желаемые товары либо картинки с примерами. Можно удалять как желания, так и целые списки. При переходе в профиль отображаются имя пользователя, электронная почта, дата регистрации, списки, в которых есть брони, и созданные списки(при включённой опции "Уведомлять ли" отображается сообщение, когда остаётся меньше недели до события, о прохождении даты события уведомляется в любом случае)
